Для запуска тестов Showcase необходимо выполнить следующие действия:

1. загрузить Selenium IDE для Chrome
2. скачать файл testshowcase.side: https://github.com/AndreiShved/showcasetests
3. открыть Chome и нажать иконку SE справа от командной строки
4. при запуске Selenuim IDE выбрать Open an existing project  и выбрать скачанный файл
5. в открывшейся программе нажать значок треугольника для запуска нужного теста или значок треугольника с горизонтальными чёрточками для запуска всех тестов
